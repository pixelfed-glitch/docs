# Configuration

## Server Settings

| .env key      | Default |  Description | Type |
| ------------- | :-----------: | :----: | :----: |
| APP_DOMAIN | NULL | The app domain | String |
| APP_NAME | "Pixelfed" | The name of your server | String |
| OPEN_REGISTRATION | true | If your server allows new account registrations | Boolean |
